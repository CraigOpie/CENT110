Programming Constructs

Input/Output

If a program produces results, output is required to display and/or save those results. Input is required to interact with the user.

Simple I/O

Simple I/O refers to console input and output.

File I/O

File I/O refers to file input and file output.

Selection Statements

Selection statements allow executing only some of the lines of a program, depending on some test condition.

Repetition Statements

Repetition statements are used to repeat lines of code.

While Loops

While loops are used when the number of repeats is variable.

For loops

For loops are used when the number of repeats is fixed.
